Quick answers

What is Project Car Build Log App?

A project car build log that stays with the car. BoostHub connects the topic to public build profiles, photos, specs, mods, proof, BoostMap, DynoRank, events, tools, and marketplace support.

Is BoostHub only for one make or model?

No. BoostHub is built for imports, domestics, euro builds, trucks, street cars, track cars, drag cars, show cars, and long-term project cars.

Is BoostHub live on iOS and Android?

Yes. BoostHub is live on both iOS and Android, with public build profiles, BoostMap, DynoRank, events, tools, and marketplace support.
