Full Feature List

Everything
BoostHub
Includes

A straight inventory of the app and web-app tools: garage profiles, proof, rankings, events, maps, groups, marketplace trust, account settings, and the utility calculators people actually use while building.

Garage

Build Profiles

Core
  • Create, edit, delete, and share garage builds from your account.
  • Save build name, owner/shop, year, make, model, vehicle type, cylinders, induction, drivetrain, transmission, and fuel type.
  • Track horsepower, torque, turbo or power adder, ECU, injectors, clutch, fuel, mod list, suspension setup, brake setup, and chassis details.
  • Set city, state or region, country, public visibility, and BoostMap-ready location details.
  • Upload build photos, gallery photos, dyno sheets, and short build videos tied to the vehicle.
  • Request verification and show verified build status where available.
Proof

Numbers and Receipts

Proof
  • Attach dyno sheets with pinch-zoom viewing.
  • Add Dragy screenshots for 1/4 mile, 1/8 mile, 0-60 MPH, and 60-130 MPH.
  • Store elapsed time, trap speed, notes, proof image, and optional track location.
  • Show race numbers including 1/4 mile, 1/8 mile, 60 foot, reaction time, and trap speed.
  • View build video playback and processing status.
  • Add photo captions, photo location, date taken, professional-photo flag, and photographer credit.
DynoRank

Leaderboards

Rank
  • Rank builds by horsepower, 1/4 mile, 1/8 mile, 60 foot, reaction time, and trap speed.
  • Rank Dragy results by 1/4 mile, 1/8 mile, 0-60 MPH, 60-130 MPH, 100-200 KM/H, and track lap categories.
  • Filter by vehicle type, cylinders, induction, drivetrain, transmission, fuel, verification, and track.
  • Open leaderboard entries directly into the related build profile.
  • Support imperial and metric unit display.
Discover

Community Feed

Social
  • Browse For You, Recent, Following, Country, and Groups discovery views.
  • See Build of the Week and public build activity.
  • Search by year, make, model, build name, owner, or username.
  • Follow builders and follow build topics like make, model, country, fuel, or drivetrain.
  • Like builds, comment on builds, hide feed items, and view why a card was shown.
  • Grouped activity keeps one active builder from flooding the feed.
Groups

Clubs and Crews

Groups
  • Create public groups for local crews, platform groups, event hosts, shops, and general communities.
  • Join, leave, and open group pages from Discover.
  • Post group updates as a member.
  • View group posts, members, member builds, hosted events, and roles.
  • Manually add garage builds to a group instead of auto-filling personal builds.
  • Share group links that open on the website behind account sign-in.
Events

Calendar and Alerts

Events
  • Browse a community calendar for meets, dyno days, cruises, track nights, and multi-day events.
  • Create events with title, optional host/organizer, venue, street address, city, region, start time, optional end time, event link, flyer images, and details.
  • RSVP as going or interested and review My RSVPs.
  • Use selected-day and upcoming-event views.
  • Tag events to saved BoostMap locations when available.
  • Use optional GPS-based nearby event alerts.
  • Share public event pages that show details and flyers without requiring login; sign-in is only needed to RSVP, create, or edit.
BoostMap

Map and Tracks

Map
  • View public build locations and saved BoostMap places on a map.
  • Use city, state, country, latitude, and longitude data tied to builds and venues.
  • Browse curated tracks, drag strips, road courses, shops, dyno locations, raceways, and venues.
  • Open map pins into related builds, tracks, or event context.
  • Submit track times with build, track, result type, elapsed time, trap speed, notes, and proof URL or file.
  • Compare track results by location and category where data exists.
Marketplace

Parts and Sellers

Market
  • Browse marketplace listings for parts, cars, services, shops, vendors, and partner inventory.
  • Filter listings by category, seller tier, trust level, and search terms.
  • Save listings, report listings, and report sellers.
  • Create listings through the marketplace flow.
  • Start seller verification and seller account setup.
  • Show seller trust signals, verification status, safety status, trust score, and storefront-style shop data.
Tools

Build Calculators

Tools
  • Basic VIN lookup.
  • Injector calculator with fuel type, BSFC, injector count, and duty cycle.
  • Wheel and tire fitment comparison with offset, poke, clearance, ride height, speedometer effect, diameter, circumference, and sidewall changes.
  • Fuel mix calculator for ethanol blend math.
  • Boost converter.
  • Speedometer correction and gear/RPM calculator.
Profiles

Identity and Trust

Profile
  • Claim a username, display name, avatar, and bio.
  • Share public profiles and open public builder pages.
  • Show profile badges including creator, launch, country, horsepower, verification, and photographer-related badges.
  • Enable automotive photographer profile fields, service area, portfolio links, booking link, Instagram link, and availability.
  • View follower and following previews.
  • Track builder momentum, build logs, photos, group memberships, and suggested next actions.
Messages

Inbox and Notifications

Inbox
  • Open the message inbox from header actions and marketplace flows.
  • Use notification badges for unread activity.
  • Receive notifications for likes, comments, follows, events, verification, account activity, and app updates.
  • Turn push notifications on or off by device.
  • Choose whether to receive app update alerts.
Web App

Website Sharing and Account Gate

Web
  • Open the signed-in web app at boosthubapp.com/app.
  • Keep builds, profiles, groups, map pages, tracks, marketplace listings, rankings, and tools behind account sign-in.
  • Open shared event links publicly on the website, including event details, location, schedule, host name when provided, event link, RSVP counts, and flyers.
  • Open shared build, group, listing, map, and track links on the website when someone is outside the mobile app.
  • Create and edit web builds, browse groups, view map and track pages, browse marketplace listings, start seller verification, run calculators, and RSVP to events.
  • Use account-required web flows so private member data is not exposed as a public preview.
Account

Settings and Safety

Account
  • Sign in, create an account, reset password, and confirm auth links.
  • Choose light, dark, or system appearance.
  • Choose imperial or metric units across tools, rankings, and build displays.
  • Send feedback and contact support, reports, partnerships, or general inquiries.
  • Report builds, comments, photos, listings, and sellers for moderation.
  • Block and unblock users, view legal/privacy pages, and delete your account and data.
